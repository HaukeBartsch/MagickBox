M-B

A computer that receives DICOM data - performs a computation - and sends you the results back.

 

Black-box recipe

  1. DICOM push from PACS/OsiriX/storescu
  2. A "bucket" processes the data (INPUT)
  3. Result (OUTPUT) is send back using DICOM push
